The Gravitational Switch Gate

One bit of computation from nothing but Newtonian gravity. Ball B (cyan) is the signal and always flies. The input bit is whether ball A (amber) is launched: absent, B coasts straight into port 0; present, a close hyperbolic flyby bends B ~54° into port 1.

The gate

Same trick as the Fredkin–Toffoli billiard-ball computer, but the elastic collision is replaced by two-body Kepler scattering. Switches plus routing give AND / OR / NOT.

The physics

Equal unit masses, G = 1, impact parameter b = 1, relative speed v = 2. Kepler predicts tan(θ/2) = G(m₀+m₁)/(bv²) = 0.5 → θ ≈ 53°. The sim measures 54.25° (the long-range tail adds a little).

The catch

Chaos eats ~constant digits of initial-condition precision per gate and gravity has no restoring force, so there is no error correction. This computer works only in the exact-real idealization — that decay is the research object.
